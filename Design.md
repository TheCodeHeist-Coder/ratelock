# Register Page

Create a fully responsive futuristic registration page using React + Tailwind CSS with a dark cyberpunk-inspired UI.

Design Requirements:
- Use a full-screen split layout.
- Left side should take around 65% width on desktop and display a glowing futuristic network/grid background.
- Right side should take around 35% width and contain the registration form.
- Overall theme: black background with neon green/teal highlights.
- Style inspiration: advanced AI infrastructure / sci-fi dashboard / cyberpunk SaaS landing page.

  - “GLOBAL MESH NETWORK ACTIVE”

- Main headline:

  - muted gray text

Technical Requirements:
- Use React functional components.
- Use Tailwind CSS utility classes only.
- Make layout fully responsive:
  - stack vertically on mobile
  - form becomes full width.
- Add smooth hover and transition animations.
- Use backdrop blur and gradient overlays.
- Add subtle glowing shadows throughout UI.
- Use modern typography similar to Inter or Space Grotesk.
- Use reusable components:
  - Navbar
  - HeroPanel
  - RegisterForm
  - Footer
  - SocialButton
- Include accessibility best practices:
  - labels
  - focus states
  - semantic HTML.
- Add animated background using CSS gradients or canvas/SVG effects.
- Ensure production-quality spacing and alignment.
